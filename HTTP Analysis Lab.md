# HTTP Analysis Lab

# Observed Fields
- Host: google.com
- User-Agent:Mozilla 5.0
- Accept-Language: text/html, application/xhtm l+xml, application/xml; q=0.9, */*; q=0.8
- HTTP Version:  HTTP/1.1

# Key Learnings
-> HTTP is unencrypted (can read contents easily)
->Useful in threat analysis for detecting malicious requests
-> User-Agent helps identify device used in attacks
